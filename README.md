# Aspect Dumper (Original Source Code)

This repository contains the **original source code** of the Aspect Dumper.

> **Note:** This is **not** a fixed, patched, or improved version. It is the original source code provided as-is.

## Usage

1. Open the following file:

```text
aspect/core/io/obfuscated.lua
```

2. Paste your obfuscated script into `obfuscated.lua`.

3. Open a terminal inside the `aspect` directory and run:

```bash
lune run 4sp3ct.luau
```

4. The output will be generated at:

```text
aspect/core/io/dumped_output.lua
```

This repository is intended to preserve the original source code of the dumper. No fixes, optimizations, or additional features have been applied.
